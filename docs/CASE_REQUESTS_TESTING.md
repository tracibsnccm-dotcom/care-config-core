# Case Requests & Activity — Manual Testing Checklist

- **As attorney:** create request on case → see it in list → thread shows initial body → status **open**
- **As RN:** open same case → see request → reply → status becomes **responded** (attorney sees it)
- **As attorney:** close request → composer disables → All Activity shows close event
- **As attorney:** reopen → message → All Activity updates
- **Refresh browser** on each step; everything persists.
